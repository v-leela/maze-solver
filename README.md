# Maze Solver

I am trying to solve mazes using some popular algorithms and further train an AI with reinforcement learning.

## Stage 1

* I am currently learning how to generate random mazes using the Recursive Backtracking (DFS) algorithm.

## Stage 2

* Created `dfs_generator.py` for generating random mazes using Recursive Backtracking.
* Before exploring additional maze generation algorithms, I want to solve the DFS-generated mazes using some popular maze-solving algorithms.
* Created `manual_maze_solver.py`, a simple maze game where the player solves the maze using the arrow keys (the goal is to reach the bottom-right cell).
